Para conectar sua máquina local a um repositório do seu GitHub, podemos:

- Ir no GitHub, criar um repositório no mesmo e pegar o link que vai aparecer no mesmo logo depois de criado.
- Ou vamos no repositório, clicamos em Code, e copiamos o link do mesmo (pode copiar o HTTPS se você ainda não sabe o que é uma chave SSH).

Já no terminal (darei um exemplo de um repositório aqui do meu GitHub mesmo):
- git remote add origin + link copiado. Exemplo:
- git remote add origin https://github.com/LucDubs/voyager1.git

Pronto, sua máquina está conectada! :)
